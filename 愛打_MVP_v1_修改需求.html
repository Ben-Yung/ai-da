Phase I MVP v1 檢討後 required changes

📅 2026-04-30 ⚡ 閃電 × 快閃軟件客 📄 MVP v1 Post-Review
1. 主畫面 Redesign P0
參考「金猴爺老虎機」風格，全面 redesign home screen。
要求：
• 加 logo（👺 emoji + 毛筆風「愛打」字樣）
• Background 用豐富 gradient / 傳統 pattern，唔好淨係黑色
• 5 張 card 改 3+2 排列，唔好迫到 full screen
• 顏色豐富：金色、紅色、暗紋裝飾
• 參考對象：金猴爺老虎機 — 街機捕魚遊戲（色彩、佈局、氛圍）
2. 口訣語音朗讀 P0
口訣顯示文字時同時用語音讀出。
要求：
• 用 Web Speech API 嘅 TTS
• Lang = zh-HK（粵語）
• 每句口訣顯示時自動朗讀
• 音效開關可控制
3. 揀紙人要用揀嗰款 P0
目前紙人係 random 出，應該用 prescreen 揀嘅款式。
要求：
• _startGame() 傳 dollId 入 gameEngine.start()
• _spawnDoll() 改用傳入嘅 dollId 取代 random
• 紙人顏色/形狀跟所選款式
4. 道具視覺化 P0
揀咗道具（拖鞋/棍棒/老虎鉗/香爐）要顯示喺 cursor/finger 位置。
要求：
• Canvas 上畫道具 emoji icon 跟住手指/鼠標
• 道具 icon 位於 cursor 上方（約 36px offset）
• 大小約 28-32px
• 冇郁動時維持顯示
5. 打中紙人要消失 + 碎裂動畫 P0
目前打中紙人只係改變移動方向，應該要碎裂消失然後重生。
要求：
• 打中 → 碎裂動畫（紙人縮小 + fade out）
• 動畫時長約 400ms（easeOut）
• 完成後 spawn 新紙人
• 狀態機：alive → dying → dead → spawn
6. 紙人名稱顯示喺身體上 P0
揀紙人時輸入名稱，名稱長註顯示喺紙人身體中央。
要求：
• Prescreen 揀紙人後輸入稱呼
• 名稱繪製喺紙人身體中央（白色字 + shadow）
• 字體大小隨紙人縮放自動調整
• 唔好放喺紙人下面
7. 拍打聲區分 P1
打小人 vs 愛的抱抱 嘅拍打聲要明顯不同。
要求：
• 打小人：重低音 thump（100-150Hz），短促有力
• 愛的抱抱：溫柔 chime（600-800Hz），soft triangle wave
• 強度隨 combo 遞增
8. 道具動作差異化 P1
唔同道具要有唔同嘅打擊效果同動畫。
要求：
• 拖鞋：扇形拍打範圍，碎片飛散
• 棍棒：直線揮擊，長條形 hit 效果
• 老虎鉗：精準點擊，集中 spark
• 香爐：圓形擴散火焰效果
• 羽毛/唇印/玫瑰（抱抱）：各自溫柔特效

金猴爺老虎機 — 街機捕魚遊戲
整體風格：色彩豐富、金色紅色調、傳統中國 decorative 元素、畫面熱鬧有質感、icon 清晰大膽。

  • 遊戲畫面：暗底但唔係死黑，有 gradient 同裝飾
  • 動畫效果：豐富但唔過度，每個互動都有 visual feedback

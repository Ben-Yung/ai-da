📋 專案概覽

將香港「打小人」傳統習俗轉化為 Web 社交娛樂平台 MVP。

打者愛也 — 兩個核心遊戲體驗：發洩式「打小人」＋ 暖心「愛的抱抱」。

口訣節奏 + 音樂同步 + 紙人命名 — 三層互動體驗。

🏷️ 紙人命名系統（最新新增）

🏷️ 打/抱之前先改名

概念：玩家揀選紙人後，可以輸入一個自訂稱呼。之後每次見到同一張紙人，都顯示佢改嘅名。

例子：

同一張「驚蟄紙人」👇

你叫佢 → 「死婆」 ✅

你朋友叫佢 → 「老闆」 ✅

兩位用戶睇到唔同名，各自獨立記錄

流程說明
① 揀紙人從預設紙人庫揀一款
② 輸入稱呼「請輸入你想打/抱嘅名：」
③ 開始遊戲遊戲中紙人顯示自訂名 + 口訣自動用人名
④ 記存下次揀同一紙人，自動顯示上次嘅名
⑤ 可修改想改名隨時重新輸入

* 口訣自動代入人名：「打你個〈名〉頭，等你一世唔出頭」

# 👺 愛打 — Beat with Love

香港傳統習俗「打小人」社交娛樂平台

**打者愛也** — 有打有愛，發洩兼祝福

## 文化背景

「打小人」是香港的非物質文化遺產，傳統於驚蟄（每年三月初五前後）舉行，透過拍打紙人來驅散霉運、鎮壓小人。最具代表性的是鵝頸橋橋底的打小人攤檔。

## 專案願景

將這個傳統習俗現代化，變成一個**社交＋娛樂**平台：
- 🎭 **文化推廣** — 讓更多人認識打小人
- 😤 **情緒出口** — 提供安全有趣的發洩方式
- 🤝 **社交連結** — 讓人們在群組中一起玩、一起笑
- ❤️ **疼愛模式** — 祝福朋友、為自己許願
